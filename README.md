# Color Pop iOS

Color Pop is a SpriteKit color-matching game. Players tap the balloon that matches the written color name while racing against a countdown timer.

## Features

- Title screen and gameplay scene
- Randomized color prompt and balloon choices
- Touch-based selection
- Score and timer tracking
- Bonus time rewards
- High-score persistence
- Particle effects for visual feedback

## Tech Stack

- Swift
- SpriteKit
- SKScene
- SKSpriteNode
- SKLabelNode
- SKAction timers
- NSKeyedArchiver persistence

## Run Locally

1. Open `Color Pop.xcodeproj` in Xcode.
2. Run the app in an iOS Simulator.
